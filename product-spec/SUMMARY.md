# Summary

Codalotl, or `codalotl` (the binary), is an AI-powered software construction system specifically for Go projects.

It is a TUI and CLI.

## Default Environment

- Runs on Mac and Linux (including WSL), in a terminal.
- Requires certain Go-related installed programs installed:
    - go, gopls, goimports, gofmt
- Requires the git binary, and that the software we're building is version controlled with git.
