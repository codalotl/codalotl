# sseclient

sseclient implements a minimal SSE client to consume HTTP APIs that stream data back. Plays well with the stdlib net/http lib.

## Dependencies

No third-party deps. Only stdlib.

## Spec

Implements the spec defined at https://html.spec.whatwg.org/multipage/server-sent-events.html
